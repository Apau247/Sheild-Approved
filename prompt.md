## Technical Requirements

### WordPress Setup
- **CMS:** WordPress 6.8+
- **Page Builder:** WPBakery Page Builder (Visual Composer)
- **Slider:** Slider Revolution 6.x
- **Plugins:**
  - Contact Form 7
  - TRX Addons (for custom shortcodes: services, testimonials, skills, pricing)
  - Essential Grid (optional, for portfolio layout)
  - Booked (optional, for appointment booking)
  - WP GDPR Compliance
  - Date Time Picker Field

### Theme Files Required
- `style.css` – Main stylesheet with custom variables
- `css/__custom.css` – Custom overrides
- `css/__colors_default.css` – Default color scheme
- `css/__colors_dark.css` – Dark mode colors
- `css/responsive.css` – Mobile responsiveness
- `js/__scripts.js` – Theme initialization and mobile menu

### Responsive Behavior
- **Desktop (>1024px):** Full layout as described
- **Tablet (768px–1024px):** 2-column grids, stacked testimonials
- **Mobile (<768px):**
  - Single column layouts
  - Hamburger menu
  - Stacked pricing cards (swiper slider)
  - Hidden top bar (optional)
  - Full-width contact form

### Performance & SEO
- Lazy loading on all images
- WebP format for photographs where possible
- Alt text on all images
- Proper heading hierarchy (single H1, logical H2–H6)
- Schema.org markup for Organization and LocalBusiness
- Open Graph meta tags
- Favicon: 32x32 and 192x192 PNG

## Notes for AI/Builder

1. **Keep all relative paths** as `wp-content/uploads/...` or `wp-content/themes/...` so they can be adapted to the final domain.
2. **Preserve all SCSS/CSS variable structures** but update color values to the new scheme.
3. **Maintain the same class names and HTML structure** (e.g., `vc_row`, `wpb_column`, `sc_layouts_*`, `trx_addons_*`) for full compatibility with the MaxCube theme and WPBakery.
4. **Replace all instances** of:
   - `Gold Shield Security` → `Iron Vault Security`
   - `gssltduk.com` → `ironvaultsecurity.co.uk`
   - `+447349601478` → `+44 20 7946 0958`
   - `info@gssltduk.com` → `IronVaultSecurity@protonmail.com`
   - `20 Farringdon St, London EC4A 4AB, UK` → `45 Bankside Lane, London EC2V 7NQ, UK`
5. **Update the JivoSite widget code** if live chat is required, or remove the script line entirely.
6. **Ensure all `mailto:` and `tel:` links** use the new contact details.
7. **Update Google Maps iframe** to reflect the new address.
8. **Maintain the dark/light scheme classes** (`scheme_dark`, `scheme_default`) as they control theme color switching.
